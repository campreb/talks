Remote Pairing with tmux

by Patrick Davey on 21 Nov 2013 at Christchurch Ruby

This talk introduces the basic functionality of tmux - creating and navigating between multiple sessions and windows within sessions.

We also show how you can use a shared tmux session to enable collaborative pairing within the same editing environment.
